Easily publish your OneNote pages to your Wordpress site!

== Description ==

The OneNote Publisher for WordPress plugin allows you to easily publish pages from your Microsoft OneNote notebooks to your WordPress sites.

The plugin utilizes the OneNote API which allows you to access any page of any OneNote notebook stored on OneDrive.

== Installation ==
**Installation**:

1. Extract the OneNote Publisher for WordPress plugin (wp-onenote-plugin.zip) to the plugins directory of the WordPress installation.
1. To activate the plugin, login to the WordPress site as an administrator and navigate to the list of plugins. Find the "OneNote Publisher for WordPress" plugin and click on the "Activate" link.

**Configuration**:

1. After installing the plugin, on the plugins list, click "Settings" for the "OneNote Publisher for Wordpress" plugin.
1. You need to setup an MSA ClientID & Client Secret. To see the step-by-step instructions for this, click on the link on the bottom of the "Settings" page that says "Click here for instructions to setup your ClientID & Secret".

== Frequently Asked Questions ==

= I can't see all of my OneNote notebooks when using this plugin. Where are they? =

This plugin only supports accessing notebooks stored on OneDrive. If you can't access your notebooks it may be because they are stored locally on your computer, OneDrive for Business, Sharepoint, or another site.
